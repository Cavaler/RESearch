Plugin exposed API reference

Numeric:
Plugin.Call("F250C12A-78E2-4ABC-A784-3FDD3156E415", NNN)
  0.. 99  = Panels menu item
100..199  = Editor menu item + 100
200..299  = Viewer menu item + 200
